# Constraints

## Allowed Technologies
- HTML5
- CSS3
- Vanilla JavaScript
- Swiper.js (carousel only)

## Forbidden Technologies
- React / Angular / Vue
- Bootstrap / Tailwind / Material UI
- Any CSS framework
- Any JS framework/library except Swiper.js

## HTML Rules
- Use semantic tags: header, section, footer, nav, article
- Use div-based layouts
- NEVER use: table, tr, td, th
- Avoid: br, hr (unless absolutely necessary)

## CSS Rules
- No inline styles (unless absolutely necessary)
- No redundant or unused CSS
- No hardcoded magic values without purpose
- Font: Open Sans (Google Fonts) ONLY

## Validation Checklist (run before EVERY change)
1. No forbidden technology introduced?
2. Design consistency maintained?
3. Layout pixel-perfect?
4. No broken browser compatibility?
5. No inline styles added unnecessarily?
6. Semantic HTML preserved?
