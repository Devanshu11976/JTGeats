## 🍕 About The Project

**JTGeats** is a courier service landing page where authentic home-cooked food is delivered to customers. This project is a **pixel-perfect** implementation of a Figma design at `1440×2029px`, built entirely with vanilla HTML, CSS, and JavaScript — no frontend frameworks.

The project features a full-screen hero section, a responsive food card grid, a Swiper.js popular items carousel, and smooth cart interactions — all matched to exact Figma specifications.

---

## ✨ Features

| Feature | Description |
|---|---|
| 🎨 **Pixel Perfect UI** | Matched to Figma specs at 1440px with exact spacing, typography & dimensions |
| 🍕 **Food Card Grid** | 3×4 responsive grid with discount badges, ratings, delivery time & cart controls |
| 🎠 **Swiper Carousel** | Popular Items section with custom green arrows & smooth sliding |
| 📱 **Responsive Design** | Mobile-first layout adapting from 1 to 4 columns across all screen sizes |
| 🛒 **Cart Interactions** | Add-to-cart buttons with quantity counter — plus/minus controls via JS |
| ⚡ **Pure Vanilla** | Zero frameworks, zero dependencies except Swiper.js — fast & lightweight |
